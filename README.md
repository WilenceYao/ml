# ml

minimal lisp interpreter written in C.

## Summary
A minimal lisp interpreter.

Implemented:
 * Grammar
 * Arithmetic expression


## HOW TO
### clone and compile

```
git@github.com:WilenceYao/ml.git
cd ml
make
```

### run

```
./ml
ml version 0.0.1
Press Ctrl+c to Exit

ml> ( + 1 3.4 )
4.4
ml>
```
